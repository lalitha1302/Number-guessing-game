#  Number Guessing Game (Using NumPy)
##  Project Description
The **Number Guessing Game** is a simple Python-based game where the computer randomly selects a number, and the player tries to guess it within a limited number of attempts.  
This project uses **NumPy** to generate random numbers efficiently.
